Hexo简介

Hexo是一款基于Node.js的静态博客框架，可以方便的生成静态页面托管在github和coding上，是搭
建博客的首选框架。大家可以进入hexo官网进行详细的查看

*1.安装git

*2.安装Nodejs

*6.将Hexo部署到Github

*8.Hexo常用命令

#Stept8： Hexo常用的命令

  • hexo c//清理缓存
  • hexo g//生成新数据
  • hexo d//部署
  • hexo s//启动本都服务概览
  • npm install hexo -g//安装Hexo
  • npm update hexo -g//升级
  • hexo init #初始化博客
  • hexo server -p 5000 #更改端口号
